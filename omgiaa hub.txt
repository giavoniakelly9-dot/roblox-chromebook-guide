  <script>
    // Simple search helper - opens selected engine with query
    function openSearch(engine) {
      const q = encodeURIComponent(document.getElementById('searchInput').value.trim());
      if (!q) return;
      let url = '';
      if (engine === 'google') {
        // Google with safe parameter
        url = 'https://www.google.com/search?q=' + q + '&safe=active';
      } else if (engine === 'ddg') {
        url = 'https://duckduckgo.com/?q=' + q;
      } else if (engine === 'wiki') {
        url = 'https://en.wikipedia.org/w/index.php?search=' + q;
      } else if (engine === 'yt') {
        url = 'https://www.youtube.com/results?search_query=' + q;
      }
      window.open(url, '_blank', 'noopener');
    }

    function doSearch() {
      // default search opens with DuckDuckGo
      openSearch('ddg');
      return false; // prevent form submit
    }

    // Notes save/load (localStorage)
    const notesEl = document.getElementById('notes');
    const saveBtn = document.getElementById('saveNotes');
    const NOTES_KEY = 'omgiaa_notes_v1';
    function loadNotes() {
      const s = localStorage.getItem(NOTES_KEY) || '';
      notesEl.value = s;
    }
    function saveNotes() {
      localStorage.setItem(NOTES_KEY, notesEl.value);
      saveBtn.textContent = 'Saved ✓';
      setTimeout(()=> saveBtn.textContent = 'Save note', 1200);
    }
    saveBtn.addEventListener('click', saveNotes);
    loadNotes();
  </script>

# omgiaaunblocked

A safe, legal hub site for quick searches and links. Designed to be hosted on GitHub Pages.

features
- pastel pink / royal blue / dark red / black theme
- search box that opens Google, DuckDuckGo, Wikipedia or YouTube
- official links to Roblox and Instagram (placeholders included for your own legal cloud URLs)
- local browser-stored notes

how to deploy on GitHub Pages
1. create a new repository on GitHub named `omgiaaunblocked` (or any name).
2. upload `index.html` and `style.css` to the repository root.
3. in the repository settings -> Pages, set the source to the `main` branch and the root folder (or enable GitHub Pages following GitHub UI).
4. within a minute or two your site will be available at `https://<your-username>.github.io/<repo-name>/`

notes about placeholders and safety
- This project intentionally does not include any code to bypass network or school restrictions.
- The "Cloud / Now.gg (placeholder)" and "Alternate (placeholder)" buttons are placeholders only. If you have a legitimate cloud gaming URL (that you are allowed to use), you can edit `index.html` and replace the placeholder href with your own link.
- If a site is blocked at your school, the right thing to do is ask an admin for permission or use approved resources.
